Workspace API Docs

Rustdoc generates each workspace crate as its own entry point. This page keeps the published API docs navigable from a single stable URL.

Main crates

Start here. These crates are the primary entry points and re-export everything you need.

  • airbender-host

    main library crate airbender_host

  • airbender-sdk

    main library crate airbender

  • cargo-airbender

    main binary crate cargo_airbender

Support crates

Lower-level building blocks consumed by the main crates. Depend on these directly only when you need fine-grained control.
